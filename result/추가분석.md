
# ⭐ **현업/포트폴리오에서 가장 많이 쓰는 금융 데이터셋 10선**

## 1) **IEEE-CIS Fraud Detection (Kaggle) — 금융 사기 데이터 끝판왕**

* 실제 대형 글로벌 금융기관 + IEEE 공동 배포
* 500만 행 수준, 변수 400개+
* 카드사기 탐지에서 가장 유명
* 모델링 + EDA + Feature Engineering 모두 가능
  👉 **포트폴리오 1등 데이터**

---

## 2) **Credit Card Fraud Detection (European Cardholders)**

* 매우 유명한 비식별화 카드사기 데이터
* PCA로 변환됐지만, 불균형 데이터 해결 사례로 최고
* 분석·시각화·머신러닝 모두 가능

---

## 3) **Home Credit Default Risk (Kaggle) — 신용평가의 표준**

* 금융기관 대출 신청자 정보 + 21개 연계 데이터
* 실제 신용평가 Feature Engineering 대표 케이스
* 부도율 예측 프로젝트 만들기 최적
  👉 **신용평가 포트폴리오 필수**

---

## 4) **Lending Club Loan Data (P2P 대출)**

* 미국 대출/이자/연체/상환 데이터
* 수십만 고객의 대출 내역 + 개인 속성
* **EDA → 신용위험 분석 → 모델링** 코스에 완전 적합

---

## 5) **Fannie Mae / Freddie Mac Mortgage Performance**

* 미국 주택담보대출(Mortgage) 실적 데이터
* 20년 단위로 수백만 건
* 금융·부동산·거시경제 분석 포트폴리오에 강력

---

## 6) **Bureau of Labor Statistics — 소비/지출 데이터 (CPI/지출패턴)**

* 경제 분석 + 금융 포트폴리오에서 응용 가능
* 물가·소비 패턴 분석 프로젝트에 좋음

---

## 7) **Quantopian / Zipline 금융 시계열 데이터**

* 주가·ETF·인덱스 시계열
* 백테스트/퀀트 분석 포트폴리오 만들기 가능
  (현재 Quantopian 종료됐지만 데이터는 GitHub에 많음)

---

## 8) **Yahoo Finance / FRED API (금융 시계열)**

* 주가/지표/거시금융
* yfinance로 쉽게 수집 가능
* 포트폴리오 최적화, 리스크 관리, 트렌드 분석 등 가능

---

## 9) **Bank Marketing Dataset (UCI)**

* 포루투갈 은행 마케팅 캠페인 데이터
* 고객 반응 예측
* EDA & Classification 실습에 인기

---

## 10) **German Credit Dataset / Australian Credit**

* 클래식한 신용평가 벤치마크
* 학습·해석·모델링에 적합
* 금융 포트폴리오에서 자주 쓰임

---

# 🔥 추천 조합 (포트폴리오 완성도 높이는 조합)

### ✔ 금융 사기(Fraud)

➡ **IEEE Fraud + Credit Card Fraud**

* EDA
* 불균형 데이터 처리(SMOTE, Focal Loss)
* 이상치 탐지
* 모델링(XGBoost, CatBoost)

### ✔ 신용평가(Credit Scoring)

➡ **Home Credit + Lending Club + German Credit**

* 연체 예측(부도율)
* 대출 리스크 평가
* Feature Importance 해석(SHAP)

### ✔ 금융 시계열(Quant/경제분석)

➡ **Yahoo Finance + FRED**

* ARIMA/LSTM 시계열 분석
* 포트폴리오 최적화(Markowitz)
* 금융 이벤트 분석(Event Study)
